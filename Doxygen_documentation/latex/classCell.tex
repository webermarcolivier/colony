\subsection{Detailed Description}
Cell able to divide and communicate with the milieu. 

This class is derived from \doxyref{CellBase}{p.}{classCellBase} and defines the cells able to live in a milieu and divide.

Remark: it has a propensities array of size nTotalChannels that contains the propensities of intracellular reactions and cell-milieu reactions.

\subsubsection{\setlength{\rightskip}{0pt plus 5cm}Array$<$int,2$>$ {\bf Cell::dnaSpecies\_\-}\hspace{0.3cm}{\tt  [private]}}\label{classCell_dc70b56b2f06995476f76b84023be877}


Array defining genetic species. 

int array of size nSpecies x nSpecies.\par
 This array defines which species are genetic material and which are proteins. It is used to define which species are halved and which species are copied during a division event. It is defined the following way:

On the diagonal, for {\tt i} == {\tt j} : \begin{TabularC}{2}
\hline


\begin{Code}\begin{verbatim} dnaSpecies_(i,i) = 0 
\end{verbatim}
\end{Code}

 &if species {\tt i} is a protein =$>$ will be halved during division event. 

\\\hline


\begin{Code}\begin{verbatim} dnaSpecies_(i,i) = 1 
\end{verbatim}
\end{Code}

 &if species {\tt i} is a protein-DNA complex =$>$ will be unbound before division event. 

\\\hline


\begin{Code}\begin{verbatim} dnaSpecies_(i,i) = -1 
\end{verbatim}
\end{Code}

 &if species {\tt i} is a DNA molecule =$>$ will be copied during division event.  \\\hline
\end{TabularC}


In the bulk matrix, for {\tt i} != {\tt j} : \begin{TabularC}{2}
\hline


\begin{Code}\begin{verbatim} dnaSpecies_(i,j) = n_ij 
\end{verbatim}
\end{Code}

 &If species {\tt i} is a protein or a DNA molecule, then 

\begin{Code}\begin{verbatim} n_ij = 0
\end{verbatim}
\end{Code}

 for all {\tt j}. If species {\tt i} is a protein-DNA complex, then {\tt n\_\-ij} define the stoichiometric vector of the unbounding reaction. In other words, it means that the protein-DNA complex is formed of: 

\begin{Code}\begin{verbatim} n_i0*species0 + n_i1*species1 + ... 
\end{verbatim}
\end{Code}

  \\\hline
\end{TabularC}
