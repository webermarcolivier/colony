\subsection{Detailed Description}
Graphics scene of the cell colony in the Qt Graphics framework. 

Contains the list of pointers to the \doxyref{GraphicsCell}{p.}{classGraphicsCell} objects, defines method to add and remove cells, defines access to a cell through the [] operator.
