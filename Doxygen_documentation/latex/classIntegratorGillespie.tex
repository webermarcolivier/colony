\subsubsection{\setlength{\rightskip}{0pt plus 5cm}void IntegratorGillespie::integrateOneStep ()}\label{classIntegratorGillespie_0b8b681cb199dfcd3ced00615b55905c}


Integrate one step in the Gillespie algorithm. 



\begin{itemize}
\item Instantiate a local reference to the array of propensities.\end{itemize}


\begin{itemize}
\item Compute the total sum of all channels propensities, 'atot'.\end{itemize}


\begin{itemize}
\item Pick up 2 uniform random numbers in [0,1), 'r1' and 'r2'.\end{itemize}


\begin{itemize}
\item Compute the time till next reaction, 'tau'.\end{itemize}


\begin{itemize}
\item Find the channel of the next reaction, 'mu'. Choose channel between all the reactions available according to the weights given by the propensities.\end{itemize}


\begin{itemize}
\item Get the time of the following division event in the whole cell collection. The next division event will happen at time 't1'.\end{itemize}


\begin{itemize}
\item If the time of next reaction is smaller than the time of next division event 't1', then\begin{itemize}
\item Apply reaction 'mu' in cell 'k' and update simulation time.\end{itemize}
\end{itemize}


If not, update simulation time to the time of next division event 't1' and divide the cell(s).
