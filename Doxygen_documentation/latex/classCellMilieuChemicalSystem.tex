\subsection{Detailed Description}
This Class defines the cell-milieu reactions like diffusion, signalling, etc. 

It is contained in a \doxyref{Cell}{p.}{classCell} object and has a pointer to the milieu \doxyref{Cell}{p.}{classCell} object of the colony. The definition of the reactions are given by the chemicalSystemCellMilieu\_\- attribute.
